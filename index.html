Welcome to

PolyMath JS is a flexible JavaScript mathematics library that supports all the functions you wish you had!
It's easy to install and use, with several installation options and many use cases.

We support various fields of math, including statistics and probability, trigonometry, calculus, arithmetic, geometry, and so much more!

Recent Updates:

PolyMath Beta has been released!
Source code is available on GitHub, and basic features have been added.
We hope to pick up the pace of updates!
